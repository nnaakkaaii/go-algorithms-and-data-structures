# ラウンドロビンスケジューリングをqueueで実装する例

ラウンドロビンスケジューリングではCPUがプロセスを順番に処理する

## 入力
入力の形式は次の通り

```
n q
name_1 tim_1
name_2 time_2
...
name_n time_n
```

最初の行に、プロセス数を表す整数とクオンタムを表す整数q
次以降のn行に、各プロセスの情報を順番に

## 出力
プロセスが完了した順に各プロセスの名前と終了時刻を出力

## サンプル
- 入力
  ```
  5 100
  p1 150
  p2 80
  p3 200
  p4 350
  p5 20
  ```
- 出力
  ```
  p2 180
  p5 400
  p1 450
  p3 550
  p4 800
  ```
